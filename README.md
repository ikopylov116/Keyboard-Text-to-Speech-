# 🎹 Виртуальная клавиатура с озвучиванием (Text-to-Speech)

Виртуальная клавиатура на Python с функцией **Text-to-Speech (TTS)**. Каждая вводимая буква автоматически озвучивается, что особенно полезно для обучения слепой печати, изучения языка и людям с ограниченными возможностями зрения.

## 🌟 Особенности

- 🔤 **Автоматическое озвучивание** — каждая буква произносится сразу при вводе
- 🔊 **Озвучивание всего текста** — нажмите **CTRL** для прослушивания всего набранного текста
- ⌨️ **Русская раскладка** — полная поддержка ЙЦУКЕН
- 🔠 **Поддержка регистра** — Caps Lock и Shift с визуальной индикацией
- 🎨 **Тёмная тема** — приятный современный интерфейс
- ⚡ **Быстрый отклик** — мгновенная обратная связь
- 🎯 **Обучение слепой печати** — идеальна для тренировки

## 📋 Требования

- Python 3.7 или выше
- Операционная система: Windows 10/11, Linux, macOS

## 📦 Установка

### Шаг 1: Установка Python

Если Python ещё не установлен:

1. Скачайте Python с официального сайта: https://www.python.org/downloads/
2. **ВАЖНО**: При установке отметьте галочку ✅ **"Add Python to PATH"**
3. Нажмите "Install Now"

### Шаг 2: Установка зависимостей

Откройте командную строку (**cmd**) **от имени администратора** и выполните команды:

#### Для Windows:

python -m pip install pynput
python -m pip install pyttsx3

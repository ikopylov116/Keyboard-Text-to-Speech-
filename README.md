Виртуальная клавиатура на Python с функцией **Text-to-Speech (TTS)**. Каждая вводимая буква автоматически озвучивается, что особенно полезно для обучения слепой печати и людям с ограниченными возможностями.

## 🌟 Особенности

- 🔤 **Автоматическое озвучивание** — каждая буква произносится сразу при вводе
- 🔊 **Озвучивание текста** — нажмите **CTRL** для прослушивания всего набранного текста
- ⌨️ **Русская раскладка** — полная поддержка ЙЦУКЕН
- 🔒 **Caps Lock и Shift** — переключение регистра с визуальной индикацией
- 🎨 **Тёмная тема** — приятный современный интерфейс
-  **Быстрый отклик** — мгновенная обратная связь
